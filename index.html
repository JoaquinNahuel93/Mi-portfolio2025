Detalles del Proyecto

Sistema de Gestión de Boletos de Cine (POO)
Objetivo del Proyecto: Este proyecto es una aplicación de escritorio desarrollada en Python y PyQt6 para la gestión de boletos de cine. El sistema permite a los usuarios seleccionar funciones, asientos y pagar boletos de manera sencilla y eficiente. Se ha desarrollado utilizando los principios de Programación Orientada a Objetos (POO), lo que permite una mayor modularidad, reutilización del código y mantenimiento a largo plazo. Características Principales: Selección de Función: Los usuarios pueden elegir entre múltiples funciones de cine disponibles en un horario determinado. Selección de Asientos: El sistema permite a los usuarios seleccionar asientos específicos dentro de la sala de cine. Visualización de Detalles: Muestra información detallada sobre la película, incluyendo título, duración, clasificación y resumen. Procesamiento de Pagos: Permite realizar pagos de manera virtual, confirmando la compra de los boletos seleccionados. Interfaz Gráfica Intuitiva: Desarrollada con PyQt6, la interfaz es sencilla y fácil de navegar, con botones, menús desplegables y alertas visuales claras.
Uso de Programación Orientada a Objetos (POO): El proyecto fue estructurado bajo los principios de la POO, lo que mejora la organización del código y facilita su mantenimiento y expansión. Se crearon clases para representar las distintas entidades del sistema (por ejemplo, Película, Boleto, Sala, Usuario), y estas clases gestionan tanto los datos como el comportamiento asociado a cada entidad. Esto permitió: Encapsulamiento: Las clases contienen los atributos y métodos relacionados, evitando la exposición innecesaria de datos. Herencia: Se utilizó para crear nuevas clases basadas en otras, permitiendo la reutilización de código y mejorando la extensibilidad del sistema. Polimorfismo: Permite que diferentes clases implementen comportamientos similares de manera flexible, sin modificar el código base. Abstracción: Se simplificaron las interacciones del usuario con el sistema, ocultando detalles complejos y presentando solo la funcionalidad esencial.
